How to install ActiveX Control     
  • In order to access the webpage, please follow these steps to download ActiveX.
  • 1.Log in the interface and Select "Install ActiveX control" .
 
  • 2.Log in again.
 
  • 3.When prompted by the File Download dialog box, click the Install button to finish the installation.
 
  • 4.To Program Files→Add/Remove Program→Check whether the installation has completed.
