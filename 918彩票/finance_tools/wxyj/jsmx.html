应缴纳所得额

应缴纳所得额=10000 - 1750 - 3500 = 4750

应纳税额=4750 x 20% - 555 = 395

税后工资

税后工资 = 10000 - 1750 - 395 = 7855

计算公式

应纳税所得额 = 工资收入金额－各项社会保险费－起征点(3500元)
应纳税额 = 应纳税所得额 x 税率－速算扣除数
税后工资= 税前工资 - 五险一金 - 应纳税额

说明

1. 如果计算的是外籍人士（包括港、澳、台），则个税起征点应设为4800元

2. 各大城市的各项保险比例或有不同，具体根据各城市制定的比例进行计算。
